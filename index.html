What is OneWall?

OneWall is a parametric customizer for vase-mode 3D-printed bins and containers. Adjust the sliders to shape the wall, texture and lip, then export an STL or download a session JSON to share.

The three height sections

Every bin is divided into a LOW section (base transition), a MID section (the main wall — vertical profile + texture) and a HIGH section (the lip / rim). Each has its own controls.

⚠ Slicer settings — required for the bottom signature

The @Amitkuzi engraving is one layer deep (0.2 mm). In your slicer set Bottom shell layers ≥ 2 (recommended: 3). With only 1 bottom shell layer the engraving punches through the floor and is visible from inside the bin. With 2+ bottom layers it's hidden inside the bottom and visible only when you flip the print over.

SlicerVase modeBottom shell
Bambu Studio Strength → Spiral vase ON, Wall loops 1 Strength → Bottom shell layers = 3
PrusaSlicer Layers → Spiral vase ON Solid layers / Bottom = 3
Cura Special Modes → Spiralize Outer Contour ON Shell → Bottom Layers = 3
