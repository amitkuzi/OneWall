## Description (paste as Markdown)

```markdown
**OneWall** is a parametric vase-mode bin / container customizer.
Move sliders, geometry rebuilds live, then export an STL with the
**@Amitkuzi** signature engraved into the underside.

The bin prints as **one nozzle-wide continuous spiral** above a few
solid bottom layers. Fast, beautiful, and the woven / fabric textures
turn the thin wall into a rigid surface.

## Tune everything

- Outer dimensions (width, depth, height, corner radius)
- Three-section height (base · mid wall · top lip)
- Vertical wall profile (linear, barrel, hourglass, ogee, bell,
  double-bulge, custom sine)
- Wall texture (smooth, ribs, rings, grid, fabric weave, basket,
  diamond lattice, helical flutes, knurl, decorative wave)
- Top lip styles (rolled, flange, bead, ribbed)
- Optional label recess on the front

## Live web preview

Try every parameter combination instantly in your browser:
**<https://amitkuzi.github.io/OneWall/>**

## ⚠ Slicer setting for the signature

> Set **Bottom solid layers ≥ 2** (3 is ideal). PrusaSlicer:
> *Print Settings → Layers → Solid layers / Bottom*.
>
> The engraving is one layer deep (0.2 mm). With 2+ bottom solid
> layers it's hidden inside the floor and **only visible from
> below**. With 1 layer it punches through.

## Recommended settings (PrusaSlicer)

| Setting | Value |
|---------|-------|
| Spiral vase | ON |
| Perimeters | 1 |
| Solid layers / Bottom | **3** |
| Solid layers / Top | 0 |
| Layer height | 0.2 mm |
| Extrusion width | 0.4 mm |
| Material | PLA / PETG |
```
